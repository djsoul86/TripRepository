Descargar el proyecto.

1. Para inicializar el front es necesario Abrir la carpeta TripRepository/Panama-Test.
2. Doble click en index.html.
3. Abrir una consola de comandos. 
4. Abrir la carpeta Viajeros/Viajeros.
5. Escribir el comando: dotnet restore.
6. Escribir el comando: dotnet build.
7. Escribir el comando: dotnet run.

Uso:

El usuario puede navegar entre las distintas pantallas, deberia poder crear, editar, eliminar y listar las diferentes entidades, viajeros, viajes y reservas. Se ha creado una tabla de paises con la lista correspondiente que es consumida por el front. Los Scripts de base de datos estan incluidos para ver la estructura de las tablas y las llaves.
